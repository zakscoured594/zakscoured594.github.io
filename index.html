⚙️ Setting Up the Extension

  1. Find the downloaded ZIP file on your computer and right-click it.
  2. Select "Extract All" and choose a folder to save the extracted files.
  3. Open Google Chrome and type chrome://extensions in the address bar, then press Enter.
  4. Locate the "Developer mode" switch in the top right corner and turn it on.
  5. Click the button that says "Load unpacked" on the top left.
  6. Navigate to your extracted folder and select it. The extension will now appear in your browser.

🛠️ System Requirements

  • Windows 10 or Windows 11.
  • Google Chrome browser installed.
  • A valid Threads account.
  • Basic internet access.

✨ How to Use

Once you install the extension, navigate to the Threads website. Click the extension icon in your browser toolbar to open the side panel. The panel displays engagement statistics for the current feed. You can sort threads by popularity to see which posts gather the most attention. This process helps you identify trends without manual calculations.
